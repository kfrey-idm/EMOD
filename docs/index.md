# Welcome to EMOD

This documentation set provides information for researchers and developers who want to modify the EMOD source code to add more functionality to the model. For example, by adding custom reports
or creating a new simulation type for modeling a disease that EMOD doesn't currently support. The docs describe how to build the EMOD executable from source code, run tests, debug errors, and more.

Please refer to other documentation sets for information on how to run EMOD as provided for malaria, HIV, or generic simulations.
